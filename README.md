## Local LLM Setup

- Install Ollama:
  https://ollama.com/download

- Install TinyLlama:

  ollama pull tinyllama:1.1b

- Run TinyLlama:

  ollama run tinyllama:1.1b

- Users with high-end hardware can run larger and more capable LLMs. Please review the model requirements before selecting a model for your system.

- More Ollama models:
  https://ollama.com/search
